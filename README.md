# Recharts Demo

A small web application showcasing chart types available in the [Recharts](https://recharts.org) library, built with React, TypeScript, and Vite.

**Live demo: [recharts-demo-lime.vercel.app](https://recharts-demo-lime.vercel.app/)**

## Charts included

| Chart       | Description                               |
| ----------- | ----------------------------------------- |
| Bar         | Monthly revenue vs expenses, grouped bars |
| Line        | Weekly active users and new sign-ups      |
| Donut / Pie | Session share by device type              |
| Radar       | Team performance across six metrics       |
| Area        | Server CPU and memory load over 24 hours  |

## Tech stack

- [React](https://react.dev/) + [Vite](https://vite.dev/)
- [TypeScript](https://www.typescriptlang.org/) (strict)
- [Recharts](https://recharts.org/)

## Development

```bash
npm install
npm run dev      # dev server at http://localhost:5173
npm run build    # production build
npm run preview  # preview the production build
```
